# TODO List - Quick Reference

> **Catatan**: Ini adalah TODO.md untuk quick reference personal.
>
> Untuk TODO list yang lebih detail dan terstruktur, gunakan sistem TODO yang sudah terintegrasi di AI assistant.
> Untuk progress tracking team-wide, lihat `docs/PROGRESS.md`.

## Cara Menggunakan TODO.md

### Fungsi:

1. **Quick Notes** - Catatan cepat untuk task sehari-hari
2. **Reminders** - Pengingat task yang perlu dilakukan segera
3. **Pending Items** - Item yang belum masuk ke sistem TODO formal

### Format:

```markdown
- [ ] Task description
- [x] Completed task
- [urgent] Task priority
- [blocked] Task yang sedang terblokir
```
